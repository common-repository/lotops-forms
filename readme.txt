Easy website contact forms that automatically update your LoTops CRM.

== Description ==

LoTops Customer Relationship Management (CRM) is a leading cloud-based application designed to help you easily grow and manage your business. With LoTops, you can manage leads and sales opportunities, grow client relationships and quickly access the key data about your business.   LoTops Forms allows you to install a contact form on your WordPress site that automatically adds submissions as Leads in your LoTops CRM.

== Installation ==

To install this plugin just follow these simple steps:

1. Upload the plugin files to the a `/wp-content/plugins/lotops-forms` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Edit the page you want to show this form in WordPress' Dashboard.
4. Enter [lotops_leads_form] as the reference tag.
5. Save and refresh to display it.

== Frequently Asked Questions ==

= Which webpage or domain will be used by LoTops? =

You can run this code from the website domain you entered in the Organization interface of LoTops. It will also work on its subdomains.

= Will this code work if I don't have an active subscription to LoTops? =

No, your LoTops account must be active for this code to work.
